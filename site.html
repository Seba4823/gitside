Git commands

Github er en slags server hvor du kan ligge dine kodning op. Du skal ind på siden "github" og der skal du lave en konto. Derefter skal du lave noget der hedder "git repository" og kalde den noget der passer til en mappe du vil kode i

Du kan installere Git på forskellige måder, som via din terminal hvor du kan indtaste en kode på din macbook eller via internetet for windows

Github giver dig disse commands som du skal bruge i terminal for at få sent det ind på deres side

Til at starte med skal du lige være i den rigtige mappe. I starten kan man bruge commandoen "ls" for at få et indblik og derefter bruger du "cd" og mappens navn som du vil ind i.

Første git command du skal bruge i mappen hedder "git init" og gør at du kan bruge git i terminalen til at starte med.

Derefter kan du oprette mapper og filer i dit program du koder i. Derefter siger du "git status" der fortæller dig status. Så burde der være en eller flere filer efter hvor meget du har lagt ind i mappen som du koder i.

Derefter bruger du commanden "git add" og så filens navn efter mellemrum. Eksempel ville være "git add grøn.html"

Så bruger man git status igen for at se om den dukker op. Det kan være at den foreksempel var rød før og nu er den grøn, sådan skal det være.

Efter det bruger vi en command der hedder "git commit -m 'indsæt hvad du har added'"

Derefter kan du sige "git log" for at se de seneste commits du har lavet

Næste og i bund og grund sidste command du skal bruge er "git push". Git push skubber selve det du har gjordt ind i det repository du lavede inde på hjemmesiden
